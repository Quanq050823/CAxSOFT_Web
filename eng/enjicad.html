Software EnjiCAD

enjiCAD is a CAD software specifically for Vietnamese users with a perpetual license. The cost of purchasing a perpetual license for enjiCAD is even lower than the annual subscription fee of other famous CAD software.

International Standard File Formats

- Fully compatible with popular file formats: *.DWG, *.DXF, *.PAT, *.SHX, ...
- Read files in formats: *.WMF, *.SAT, *.3DS, ...
- Export files in formats: *.DWG, *.DWF, *.DWFX, *.WMF Export DWF 2D & 3D.

Low System Requirements

- Compact installer <200Mb, runs smoothly, does not require high computer specifications.

- RAM: 32 bit - 3GB or more / 64 bit - 4GB or more.
